# FUS_AGENCY_APP

### 🚀 Quick Start
1. Save all files in the same folder
2. Open `login.html` in a browser
3. Create an account or continue as guest
4. Book services and manage bookings
5. Visit `admin.html` for management

### 📱 PWA Installation
- **Android**: Open in Chrome → Tap "Install App"
- **iOS**: Open in Safari → Share → Add to Home Screen

### 🔒 Data Storage
All data is stored locally in the browser using localStorage:
- `fusUser`: User profile
- `fusUsers`: All registered users
- `fusBookings`: All bookings
- `fusNotifications`: Notification history

### 🚀 Deployment
**Option 1: Netlify (Free)**
1. Create a Netlify account
2. Drag and drop the folder
3. Your app is live!

**Option 2: GitHub Pages**
1. Push to GitHub
2. Enable GitHub Pages
3. Access via `username.github.io/repo`

**Option 3: Vercel (Free)**
1. Install Vercel CLI
2. Run `vercel` in the folder
3. Follow the prompts
